# STAT_325-Project-
STAT 325 is a statistics class at Case Western Reserve University in which it focuses on Data Analysis and Linear Regression Models.

My project analyzed how immunization coverage affects life expectancy across developed and developing countries using WHO and UN datasets. I used tools like multiple linear regression, Box-Cox transformation, variance inflation factor (VIF) for multicollinearity, and residual diagnostics to validate the model. I also worked with data cleaning, variable selection, and statistical analysis techniques. The results showed that vaccines significantly improve life expectancy in developing countries, while health expenditure is the main driver in developed countries.
